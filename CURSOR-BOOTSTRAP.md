**Правила экономии токенов и выбора моделей**

- По умолчанию всегда работай в **Auto** mode.
- Для 80–90% задач (обычная разработка, правки, небольшие рефакторинги) используй быстрые модели:
**Composer 2**, **Kimi K2.5**, **Gemini 3 Flash**, **Claude 4.6 Sonnet** или **GPT-5.3 Codex**.

  - Используй только точечный контекст (@файлы, @папки, @codebase).
  - Предпочитай Skills и Hooks вместо длинных объяснений.
  - Делай короткие сессии.
  - Никогда не повторяй правила — ссылайся на `@.cursor/rules/`.
  - Следи за preCompact hook — если контекст >70%, автоматически сокращай.
